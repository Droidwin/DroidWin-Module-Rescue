# Changelog

## v1.0

Initial public release.

### Features

- Disable Module
- Remove Module
- Disable All Modules
- View Installed Modules
- Reboot System
- Interactive terminal interface
- Confirmation prompts for destructive actions
- Module display name and folder name display
- Module count display
- Invalid selection handling
- Data partition availability check

### Supported Recovery

- TWRP Recovery
- OrangeFox Recovery
